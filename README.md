# evo2026-codes

Welcome to the `evo2026-codes` repository! This repository contains a diverse collection of data science projects, statistical hypothesis tests, machine learning models, and general Python programming tasks implemented in Jupyter Notebooks.

## 📁 Repository Contents

### 📊 Data Analysis & Foundations
- **`Numpy_pandas_tasks.ipynb`**: Fundamental tasks and exercises utilizing the `numpy` and `pandas` libraries for data manipulation.
- **`data_analysis.ipynb`**: General exploratory data analysis scripts.
- **`DA11_Python_Test.ipynb`**: Python assessments and tests for data analysis.
- **`OOP_Data_Analysis.ipynb`**: Applying Object-Oriented Programming (OOP) concepts to data analysis workflows.
- **`SQL_connection_with_Pandas.ipynb`**: Methods for connecting to SQL databases and querying directly into Pandas DataFrames.

### 🧪 Statistical Hypothesis Testing
A significant portion of this repository is dedicated to statistical testing, specifically applied to real-world scenarios like UFC fights:
- **`OneSampleProportionZtest.ipynb`**: Z-Tests for single sample proportions.
- **`OneSample_Ttest.ipynb`**: Implementations of one-sample T-Tests.
- **`TwoSampleZTest.ipynb`**: Statistical comparisons between two independent sample means.
- **`UFC_Strike_HypothesisTesting.ipynb`**: Hypothesis testing applied to Ultimate Fighting Championship (UFC) striking data.
- **`UFC_TD_Average_Accuracy_HypothesisTesting_final.ipynb`**: Analysis and hypothesis testing on UFC Takedown (TD) accuracy.

### 🤖 Machine Learning
- **`014-multiclass-classification.ipynb`**: Deep dives into multi-class classification predictive modeling.

### 🐍 Python Basics & Utilities
- **`WhileLoop.ipynb`**: Beginner-friendly explanations and tasks regarding Python `while` loops.
- **`4_Errors.ipynb`**: Understanding and handling errors/exceptions in numerical data or code.

## 🚀 Getting Started

To run these notebooks locally, you will need a Python environment equipped with standard data science libraries.

### Prerequisites

You can use `pip` to install the fundamental components (assuming you have Python installed):

```bash
pip install jupyter notebook pandas numpy scikit-learn matplotlib seaborn sqlalchemy
```

### Running the Notebooks

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/sita-aghasoy33/evo2026-codes.git
   ```
2. Navigate into the repository directory:
   ```bash
   cd evo2026-codes
   ```
3. Start the Jupyter Notebook server:
   ```bash
   jupyter notebook
   ```
4. Click on any `.ipynb` file in the web interface to begin exploring the code and analysis!
